# populism-shiny
This Shiny App visualizes causal estimates of the effects of populist leaders on economic growth and distributional outcomes (e.g., inequality) using the traditional synthetic control method (SCM) and the augmented synthetic control method (ASCM), where the outcome model employs matrix completion.

To run this Shiny app:
\begin{itemize}
    \item Install R (if not already): https://www.r-project.org/
    \item Install the Shiny package: In R, run install.packages("shiny")
    \item Open R and set your working directory to the Populism-Shiny-App folder
    \item Run: shiny::runApp(launch.browser = TRUE) The app will open automatically in your web browser.
  \end{itemize}
