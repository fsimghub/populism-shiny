To run this Shiny app:
1. Install R (if not already): https://www.r-project.org/
2. Install the Shiny package: In R, run install.packages("shiny")
3. Open R and set your working directory to the Populism-Shiny-App folder
4. Run: shiny::runApp(launch.browser = TRUE)
The app will open automatically in your web browser.
